縦横両方を指定したいとき：
  python resize_img.py (ヨコ) (タテ)

横だけを指定して縦横比を維持したいとき
　python resize_img.py (ヨコ)

例１ ヨコ700px, タテ400pxにリサイズ：
  python resize_img.py 700 400

例２ ヨコ500pxで縦横比を維持：
  python resize_img.py 500

注意：
  画像は上書きされるので、元画像を残しておきたいときはバックアップを取りましょう

参考：
  記事内はヨコ700
  ヘッダーの長方形はヨコ500
  ヘッダーの円アイコンはヨコ200,タテ200
